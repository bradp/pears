## Available commands

- `yarn run build`
- `yarn run build:css`
- `yarn run build:css:prod`
- `yarn run build:hugo`
- `yarn run build:prod`
- `yarn run format`
- `yarn run format:check`
- `yarn run postinstall`
- `yarn run lint`
- `yarn run lint:css`
- `yarn run lint:css:fix`
- `yarn run start`
